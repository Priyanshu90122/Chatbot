AI Placement Prep Kit

AI Placement Prep Kit is an interactive web-based platform designed to help students prepare for technical placements by practicing interviews, quizzes, and logical thinking exercises in one place.

Features
Mock Interview Module

Unlimited interview-style questions

Random question generation on each attempt

Covers DSA, OOP, OS, DBMS, CN, and System Design

Text-based answers for self-evaluation

Quiz Engine

Multiple-choice questions with meaningful statements

Randomized questions on every attempt

Instant feedback with correct answers

Unlimited practice

Chatbot

Interactive chatbot for placement-related queries

Provides quick explanations of core CS concepts

Useful for last-minute revisions

Logic Games

Guess the Output game for JavaScript fundamentals

Tic Tac Toe game with real-time winner detection

Enhances logical and analytical thinking

User Interface

Clean and modern UI using Tailwind CSS

Glassmorphism design with smooth animations

Fully responsive for different screen sizes

Tech Stack

Frontend: HTML, Tailwind CSS

Programming Language: JavaScript

UI Design: Glassmorphism and Gradient-based styling
