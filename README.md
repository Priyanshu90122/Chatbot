# AI Placement Prep Kit

AI Placement Prep Kit is a web-based platform for technical placement preparation featuring mock interviews, quizzes, a chatbot, and logic-based games.

# Features

Mock interviews with unlimited, randomized questions

Quiz engine with instant feedback

Chatbot for CS and placement queries

Logic games including Guess the Output and Tic Tac Toe

Tech Stack

HTML, Tailwind CSS

JavaScript

How to Run

Clone the repository

Open index.html in a browser
